# Todo App  

A simple yet dynamic **Todo List application** built with **HTML, CSS, and JavaScript**.  
This project demonstrates DOM manipulation, event handling, form validation, and interactive UI features.  

---

## Features  

- **Add new todos** via form submission  
- **Remove todos** when marked as completed  
- **Check off tasks** with a strike-through effect   
- **Parent–child–sibling DOM navigation** (clicking the todo highlights its tag)   
- **Class and style manipulation** (e.g., toggle `completed` state)  
- **BOM usage** (update document title and window alerts)  

---

## Tech Stack  

- **HTML** (semantic structure, form validation)  
- **CSS** (flexbox layout, custom styles, hover states)  
- **JavaScript** (DOM, events, classList)
